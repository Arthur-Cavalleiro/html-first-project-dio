  1. Como criar um arquivo

    Voltar

    Para criar um arquivo clique com o mouse direito na área de trabalho e selecione a opção Novo e depois selecione o tipo dele.

  2. Selecionando o arquivo

    Voltar

    Para selecionar o arquivo, clique com o mouse esquerdo nele, porém se não sabe onde esta o arquivo é

    possivel pesquisar o arquivo na barra de procura do windows na parte inferiror esquerda da tela.

  3. Mover um arquivo

    Voltar Voltar

    Para Mover ele de diretório, primeiro localize o arquivo como ensinado no passo anterior *, depois clique no arquivo e escolha entre apertar
    ctrl+x
    ou segurar e arrastar o arquivo.
